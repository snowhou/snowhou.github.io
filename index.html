三、docker 常用操作

1. container 的创建、查看和删除

  • 查看 docker 主机已有的 docker 镜像

    docker images

  • 查看 dokcer 主机上已经创建的容器

​ docker ps -a -q

​ -a 表示列出所有容器（包括停止运行的容器），否则只会列出运行中的容器。

​ -q 表示只返回容器 ID 信息，其它容器信息（如状态、对应的镜像等）不显示。

  • 由镜像生成容器

​ sudo nvidia-docker run -it --ipc=host -p 8088:22 \

​ --name container-name -v /your/path:/data which-image /bin/bash

​ -it 表示创建之后，马上进入交互模式

​ -p 8088:22 端口映射

​ -v /your/path:/data 路径映射

  • 由容器生成镜像

​ docker commit -m "tag" -a="user" container-id aaa/bbb:ccc`

  • 删除镜像

​ docker rmi -f container-id

  • 删除容器

​ docker rm container-id

2. container 的启动、进入、退出

  • 启动 & 进入 (二选一)

    docker start [-i] container-id -i 表示启动后直接进入交互模式

    docker attach container-id

  • 退出

    退出，保持容器运行： CTRL+Q+P

    退出，停止容器：CTRL+D / 输入 exit再回车(enter)

  • 停止、重启

    docker stop container-id docker restart container-id
